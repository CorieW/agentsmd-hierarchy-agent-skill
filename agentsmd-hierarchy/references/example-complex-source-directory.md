# Example: Source Directory

Use this example when a source directory has local architecture and generated-artifact rules.

```md
# packages/front/src

## Rules

- Keep route modules focused on route composition and move reusable UI or data logic into feature modules.
- Regenerate generated route artifacts with the router tooling instead of editing them by hand.
- Put single-file API or styling caveats in comments at the top of the affected file.
```
